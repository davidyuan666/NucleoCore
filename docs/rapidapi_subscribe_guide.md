# RapidAPI 订阅指南

## 方法 1：直接订阅链接

### Twitter135 API（推荐，免费 500 次/月）

1. **访问订阅页面：**
   ```
   https://rapidapi.com/twitter135/api/twitter135/pricing
   ```

2. **查找订阅按钮：**
   - 页面会显示不同的定价计划
   - 找到 "Basic" 或 "Free" 计划
   - 点击 "Subscribe" 按钮

3. **如果看不到按钮：**
   - 确保已登录 RapidAPI 账号
   - 刷新页面
   - 或者点击页面右上角的 "Subscribe to Test"

---

## 方法 2：从 API 主页订阅

### 步骤：

1. **访问 API 主页：**
   ```
   https://rapidapi.com/twitter135/api/twitter135
   ```

2. **找到订阅按钮位置：**
   - 页面顶部右侧有一个蓝色按钮
   - 按钮文字可能是：
     - "Subscribe to Test"
     - "Test Endpoint"
     - "Pricing"

3. **点击后会跳转到定价页面**

---

## 方法 3：通过测试端点订阅

1. **在 API 页面找到 "Endpoints" 标签**

2. **选择任意一个端点（如 Search）**

3. **点击右侧的 "Test Endpoint" 按钮**

4. **会弹出提示要求订阅**

5. **点击 "Subscribe" 完成订阅**

---

## 常见问题

### Q: 找不到订阅按钮？
A: 可能的原因：
- 未登录 RapidAPI 账号
- 浏览器缓存问题，尝试清除缓存
- 已经订阅过了

### Q: 如何确认已订阅？
A:
1. 访问：https://rapidapi.com/developer/apps
2. 查看 "My Subscriptions"
3. 应该能看到已订阅的 API 列表

### Q: 免费计划有什么限制？
A:
- Twitter135: 500 次请求/月
- 超过后需要升级或等待下月重置
